## Lessons added today (2026-05-01 LATE / 2026-05-02 EARLY)

1. **Trailing stop wrong default for fade strategies.** `TrailingStepR=1.0` on a fade locks breakeven before the unwind completes. Fades need NO trail (let SL hold) OR very wide activation (3-5R).
2. **+DI/-DI direction filter ≠ trend direction.** On retracements in an uptrend, +DI/-DI flip locally — fires shorts at the moment book says to BUY pullbacks. Use longer-EMA-vs-price for trend context instead.
3. **Book's "ADX reset after win" rule** (Step 6) creates counter-trend bias on persistent secular trends. Default OFF for modern FX/CFD.
4. **Bootstrap MC misleads on low-N samples**, but the WR consistency across assets IS the curve-fit test (not just final return). 40-47% WR across 3 different assets = real edge mechanism.
5. **Don't add the same risk-control template to every strategy.** Trend-followers and fade strategies want different exit rules. v0.2/v0.3 risk templates need to be archetype-specific, not universal.
